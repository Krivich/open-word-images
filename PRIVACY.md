# Privacy Policy

## API Keys

- **OpenAI API Key**: Entered by users in the browser, never stored or transmitted to our servers. Used only for direct API calls to OpenAI from your browser.
- **GitHub Token** (if contributing): Used only for GitHub API calls from your browser. Not stored.

## Data Collection

This project does **not** collect, store, or process any personal data.

## Cookies

No cookies are used.

## Third-Party Services

- **OpenAI**: API calls for image generation (privacy policy: https://openai.com/privacy)
- **GitHub Pages**: Hosting (privacy policy: https://docs.github.com/en/site-policy/privacy-policies)
- **wordfreq-en-25000**: Dictionary data (CC-BY-SA 4.0)

## Security

All API calls are made directly from your browser to the respective services. No intermediate servers are involved.
